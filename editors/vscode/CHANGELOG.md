# Changelog

## 0.1.0

- Syntax highlighting for Langz (.lz) files
- LSP client integration (hover docs, diagnostics)
- Support for all builtins: I/O, file ops, system, string, networking, date/time
- String method highlighting (replace, contains, starts_with, ends_with, split, join, length)
- Pipe operator (`|>`) and compound assignment (`+=`, `-=`, `*=`, `/=`) highlighting
- String interpolation highlighting (`{var}` inside strings)
